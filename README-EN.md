# Qieman-MCP

且慢投顾 MCP Server - 智能投资顾问服务

基于模型上下文协议（MCP）的专业基金分析与资产配置工具

## 特性

- 🎯 资产与家庭财务分析
- 💼 基金与组合风险诊断
- 🔍 智能基金信息查询
- 📈 专业投资分析工具
- 🛠️ 数据可视化与报告生成

详细文档请查看 [README.md](./README.md)

## 快速开始

```bash
git clone https://github.com/Chris1Wang3/Qieman-MCP.git
cd Qieman-MCP
npm install
```
